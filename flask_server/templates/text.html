(≧∇≦)ﾉ 用户管理接口

POST
/api/login
微信小程序用户登录与注册

请求参数:

  • code - 微信登录凭证
  • register - 操作类型 (0=自检, 1=注册)
  • nickname - 用户昵称 (注册时传)
  • avatarUrl - 用户头像URL (注册时传)
GET
/api/user/<openid>
根据openid查询单个用户信息

URL参数:

  • openid - 用户唯一标识符
GET
/api/users
查询数据库中所有注册用户
DELETE
/api/delete_user/<user_id>
根据用户id删除用户

URL参数:

  • user_id - 用户数据库ID

(｡♥‿♥｡) 收藏功能接口

GET
/api/get_favorites
获取用户收藏列表

查询参数:

  • openid - 用户唯一标识符
POST
/api/add_favorite
添加音乐收藏

请求参数:

  • openid - 用户唯一标识符
  • music_id - 音乐ID
  • music_name - 音乐名称
  • music_author - 音乐作者 (可选)
POST
/api/remove_favorite
取消音乐收藏

请求参数:

  • openid - 用户唯一标识符
  • music_id - 音乐ID

(ง •_•)ง 排行榜接口

GET
/api/get_rank
获取游戏排行榜

查询参数:

  • mode - 游戏模式 (默认: single)
  • limit - 返回数量 (默认: 50)
POST
/api/upload_rank
上传游戏成绩到排行榜

请求参数:

  • openid - 用户唯一标识符
  • nickname - 用户昵称
  • avatar_url - 用户头像URL
  • mode - 游戏模式
  • score - 游戏分数
